## v0.2.1 - 货车流程增强与异步通知优化

`v0.2.1` 重点补强了货车搜索、货车详情识别、自动分享、挖掘机识别与异步通知链路，适合作为新的公开发布版本。

### 主要更新

- 新增目标货车自动分享功能
  - 命中目标后可自动点击详情页 `分享`
  - 自动选择 `R4 & R5`
  - 自动点击确认分享弹窗中的 `分享`
  - 分享后继续搜索下一辆货车
- OpenClaw 通知统一改为异步发送
  - 启动通知异步发送
  - 挖掘机通知异步发送
  - 不再阻塞主线程
- 货车列表识别与重试策略优化
  - 无效列表支持逐步放宽条件重试
  - 重试后仍无效时可按当前列表继续遍历
  - 货车列表无效与空列表逻辑更加统一
- 货车详情流程优化
  - 去掉多次详情打开重试
  - 详情页直接识别 `UR碎片`
  - 战力核实日志更明确
- 战力 OCR 改进
  - 优先在货车详情浮层中定位战力图标
  - 再对战力图标右侧数值做 OCR
  - 后备方案仍保留
- 刷新按钮识别增强
  - 仍以模板匹配为主
  - 保留蓝色区域与边界推测兜底
- 车站定位与地图处理优化
  - 地图缩小幅度增强
  - 平移地图后等待再识别车站图标
- 挖掘机识别进一步收紧
  - 优化误报控制
  - 连续帧确认后再认定
- README 完整重写，文档与当前源码行为同步

### 热键说明

- `F12`：暂停 / 恢复实时监控
- `F5`：从基地重新定位车站并开始货车搜索
- `F6`：货车搜索中暂停 / 继续，或跳过当前目标货车
- `F2`：在鼠标当前位置开启 / 停止极速连点
- `Ctrl-C`：退出程序
